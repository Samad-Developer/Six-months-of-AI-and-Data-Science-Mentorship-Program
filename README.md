# All About git

1. Initialize git repository
   ```
   git init
   ```
2. Add changes to the stage changes
   ```
    git add <path_to_fil>
    or
    git add .
   ```
3. commit changes if you are done with changes
   ```
   git commit -m "commit message"
   ```
4. to sync changes use this
   ```
   git push
   ```
